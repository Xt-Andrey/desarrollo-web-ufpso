## 📌 Parte 7: Actividad Práctica
Reto: Crear un Mini Catálogo de Productos Componentizado
Deberán construir una página web que muestre un catálogo de productos reutilizando componentes.

# Paso 1 — Componentes estructurales
Crear:
Header
Footer
y cargarlos dinámicamente en la página principal.

# Paso 2 — Lista dinámica de productos
Crear un template para mostrar productos.
Los productos se cargarán desde un array en JavaScript.

# Paso 3 — Crear un Web Component
Construir un componente:
<product-card>
Este componente debe recibir atributos:
nombre
precio
descripción

# Paso 4 — Integración
Mostrar un catálogo con mínimo 3 productos utilizando el componente creado.
